## 프로젝트 설치 및 실행
1. git clone
```bash
git clone https://github.com/KakaoTech-team20/MoreBurger_BE.git
cd MoreBurger_BE
```

2. node 런타임 환경 설치
https://nodejs.org/en/download/package-manager/current

3. npm 패키지 설치
```bash
npm install
```

4. mysql 설치
https://dev.mysql.com/downloads/

5. index.js 실행
```bash
node index.js
```
